📝 ## The Blog

🚀 ## Prerequisites
- Node.js 18+
- npm or yarn

## Installation

1. Clone the repository

`git clone <repository-url>`
`cd the-blog`

2. Install dependencies

`npm install`

4. Run the server

`npm run dev`
